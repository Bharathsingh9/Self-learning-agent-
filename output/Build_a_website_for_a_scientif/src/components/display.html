html Scientific Calculator
This HTML file is a basic structure for a scientific calculator. The display of the calculator will be updated dynamically using JavaScript when user interacts with the calculator. The CSS rules are used to style the calculator, making it visually appealing. The JavaScript code in `script.js` will handle the calculation and update the display accordingly. The calculator layout is divided into rows, which contain buttons for digits, operators, and a clear button. The `display` div will show the current calculation.
